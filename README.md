# 🚲 공공 자전거 이용 패턴 분석

> 공공 자전거(따릉이) 이용 데이터를 분석해 수요 예측 패턴을 탐색한 프로젝트

## 분석 목적
시간대, 요일, 날씨 조건에 따른 자전거 이용량 변화를 분석하여
**피크 타임 수요 예측과 배치 최적화 인사이트**를 도출합니다.

## 주요 분석 내용
- 시간대별 / 요일별 대여 건수 분포
- 날씨(기온/강수) 조건과 이용량 상관관계
- 계절별 이용 패턴 비교

## 기술 스택
`Python` `Pandas` `Matplotlib` `Seaborn` `Jupyter Notebook`

> *(분석 후 발견한 인사이트 작성)*
